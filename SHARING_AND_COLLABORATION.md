# Как поделиться кодом с другим программистом

Вы задали отличный вопрос! Совместная работа — ключ к решению сложных проблем. Вот пошаговая инструкция, как скачать код из Firebase Studio и загрузить его на GitHub.

## Шаг 1: Скачайте код проекта (ZIP-архив)

В интерфейсе Firebase Studio функция скачивания находится в панели проводника (Explorer) слева.

**Точная инструкция (на основе вашего экрана):**

1.  **Откройте Explorer:** Нажмите на самую верхнюю иконку в левом узком меню (иконка с двумя листочками бумаги). У вас откроется список всех файлов проекта.
2.  **Нажмите правой кнопкой мыши:** Найдите в списке самую верхнюю папку (корень проекта). Обычно она называется `ValutaBot`.
3.  **Выберите "Download" (Загрузить):** В контекстном меню выберите пункт **"Download..."**.
4.  **Дождитесь уведомления:** Внизу справа появится синее окошко "Zipping folder". Когда оно дойдет до 100%, браузер начнет скачивание.

**Где искать скачанный файл?**
- Проверьте папку **"Загрузки" (Downloads)** на вашем компьютере.
- Нажмите **Ctrl + J** в браузере, чтобы открыть историю загрузок и увидеть файл там.

## Шаг 2: Установите Git (если его нет)

Если на вашем компьютере не установлен Git, скачайте и установите его с официального сайта: [git-scm.com](https://git-scm.com/).

## Шаг 3: Создайте локальный Git-репозиторий

1.  Откройте терминал (Terminal) или командную строку на вашем компьютере.
2.  Перейдите в папку, куда вы распаковали скачанный ZIP-архив:
    ```bash
    cd путь/к/вашей/папке
    ```
3.  Инициализируйте репозиторий и сделайте первый коммит:
    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    ```

## Шаг 4: Загрузите код на GitHub

1.  Создайте новый пустой репозиторий на [GitHub.com](https://github.com/).
2.  Свяжите локальную папку с GitHub (команды будут на странице вашего нового репозитория):
    ```bash
    git remote add origin https://github.com/ваш-юзернейм/имя-репозитория.git
    git branch -M main
    git push -u origin main
    ```

## Шаг 5: Добавьте соавтора

1.  В репозитории на GitHub перейдите в **Settings** -> **Collaborators**.
2.  Нажмите **Add people** и пригласите вашего коллегу по его GitHub-логину или email.

Теперь ваш код доступен для совместной работы и надежно сохранен!
