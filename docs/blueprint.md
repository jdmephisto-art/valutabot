# **App Name**: ValutaBot

## Core Features:

- Currency Data Fetching: Fetch real-time currency exchange rates from a reliable API.
- Telegram Integration: Enable interaction with a Telegram bot to receive currency updates.
- Currency Conversion Tool: Allow users to convert between different currencies using current exchange rates.
- Personalized Notifications: Allow users to set up custom alerts for specific currency pairs and threshold values, delivered via Telegram.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) for trustworthiness and stability.
- Background color: Light gray (#F0F4F7), for a clean user interface.
- Accent color: Teal (#00BFA6), for highlighting important information such as exchange rates and alerts.
- Body and headline font: 'Inter' (sans-serif) for clear readability in the Telegram interface.
- Use minimalistic and universal currency icons to visually represent currencies.
- Optimize the layout for the Telegram interface for easy readability on small screens.
- Implement subtle animations, such as smooth transitions when currency rates update.
